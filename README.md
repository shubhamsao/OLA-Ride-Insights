# OLA Ride Insights 🚗

## Project Overview
Analysis of OLA ride-sharing data to extract business insights.

## Domain
Ride-Sharing & Mobility Analytics

## Tools Used
- Python (Pandas, NumPy)
- MySQL 8.0
- Power BI Desktop
- Streamlit

## Project Structure
- `data/` — Raw and cleaned datasets
- `sql/` — All 10 SQL queries
- `powerbi/` — Power BI dashboard
- `streamlit_app/` — Interactive web application
- `screenshots/` — Dashboard and app screenshots

## How to Run
1. Install libraries: `pip install -r requirements.txt`
2. Run Streamlit app: `streamlit run streamlit_app/app.py`

## Key Insights
- 62% rides are successful
- Total revenue: ₹3,50,80,467
- Auto has lowest avg distance (6.24 km)
- Vehicle Breakdown is top incomplete ride reason
